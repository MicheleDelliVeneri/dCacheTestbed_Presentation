Distributed Storage Systems

The basic architectural components are distributed among nodes (servers)  

This very simple architecture can be expanded by replicating node components in order to improve its performance or data availability and to avoid bottlenecks

dCache

Cell:

dCache basic architectural building block

Each cell performs a specific task. High-level tasks are obtained through the interaction of multiple cells

Domain

Container for running cells. Runs its own JVM instance which it cannot share with other domains.

Domain = JVM + Functionalities to run cells

Domains are split among nodes and communicate through TCP (Transmission Control Protocol)
